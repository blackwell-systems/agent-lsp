## Interface Contracts

### PrepareCallHierarchy

LSP client method. Sends textDocument/prepareCallHierarchy and returns typed
CallHierarchyItem slice. Returns empty slice if server does not support
callHierarchyProvider. Unmarshals result via json.Unmarshal into []types.CallHierarchyItem.

```
func (c *LSPClient) PrepareCallHierarchy(ctx context.Context, uri string, pos types.Position) ([]types.CallHierarchyItem, error)
```

### GetIncomingCalls

LSP client method. Sends callHierarchy/incomingCalls with the given CallHierarchyItem
and returns typed CallHierarchyIncomingCall slice. The item parameter is a typed
CallHierarchyItem from PrepareCallHierarchy; it marshals cleanly into request params.
No capability check needed -- PrepareCallHierarchy guards entry.

```
func (c *LSPClient) GetIncomingCalls(ctx context.Context, item types.CallHierarchyItem) ([]types.CallHierarchyIncomingCall, error)
```

### GetOutgoingCalls

LSP client method. Sends callHierarchy/outgoingCalls with the given CallHierarchyItem.
Same contract as GetIncomingCalls.

```
func (c *LSPClient) GetOutgoingCalls(ctx context.Context, item types.CallHierarchyItem) ([]types.CallHierarchyOutgoingCall, error)
```

### HandleCallHierarchy

Single MCP tool handler for call hierarchy. Extracts file_path/line/column/direction
from args. Direction must be "incoming", "outgoing", or "both" (default "both").
Opens the document via WithDocument, calls PrepareCallHierarchy, then calls
GetIncomingCalls and/or GetOutgoingCalls based on direction. Returns JSON with keys
"items" (prepared hierarchy items), "incoming" (omitted when direction is "outgoing"),
"outgoing" (omitted when direction is "incoming"). Returns a human-readable "no item
found" message when prepare returns empty.

```
func HandleCallHierarchy(ctx context.Context, client *lsp.LSPClient, args map[string]interface{}) (types.ToolResult, error)
```

### fuzzyPositionFallback

Package-internal helper that retries a position-based LSP lookup using workspace
symbol search when the direct lookup returns an empty result. Extracts a symbol name
from hover at the original position, searches workspace symbols for that name, retries
lookupFn at each candidate position, and returns the first non-empty result set.
Returns an empty slice (not an error) if all fallback attempts also return empty.
line and col are 1-indexed (tool convention); the function converts internally.

```
func fuzzyPositionFallback(
    ctx context.Context,
    client *lsp.LSPClient,
    fileURI string,
    line, col int,
    lookupFn func(pos types.Position) ([]types.Location, error),
) ([]types.Location, error)
```
